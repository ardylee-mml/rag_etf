Troubleshooting

If you're having trouble accessing the application:

  1. Make sure the Vite server is running (npm run client)
  2. Make sure the API server is running (./start-server.sh)
  3. Make sure the proxy server is running (node simple-proxy.js)
  4. Try clearing your browser cache or using an incognito window
